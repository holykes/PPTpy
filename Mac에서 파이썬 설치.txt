*맥에서 파이썬 설치
맥에는 파이썬이 기본 설치되어 있다.
1.터미널 창을 연다.
2.터미널 창에 "사용자명$" 가 표시된다
3.python --version 을 입력해서 버젼을 확인할 수 있다.
4.다운받은 파이썬 설치한다.
  계속-계속-계속-동의-설치-사용자 이름과 암호를 입력하면 설치가 된다.
5.설치가 완료되고 표시되는 창에서 왼쪽에 보면 "응용 프로그램"이 보인다.
  응용 프로그램을 클릭하면 설치된게 보인다.
6.터미널 창에 python3 --version 을 입력해서 현재 설치한 버젼을 확인할 수 있다.
  참고) python --version 을 입력하면 원래 설치된 버젼이 표시된다
7.터미널 창에 python3 을 입력하고 엔터치면 파이썬 프롬프트(>>>)가 표시된다.
8.프롬프트에 exit()을 입력하면 파이썬을 종료할 수 있다.
  >>>exit()
9.다운받은 vsc 파일을 더블클릭하면 파일이 하나 생긴다. 실행하려면 더블클릭하면 된다.
  참고) 메뉴에서 "이동"을 클릭하고 하위 메뉴 중 "응용 프로그램"을 선택하고 창이 뜨면 visual studio code를 이동해서 등록하면 좋다.
10.설치한 파이썬과 VSC를 연결해야 한다.
11.VSC 상단 메뉴에서 "view"-"Command palette"클릭하고 shell command를 입력하면 표시되는 것 중 "install code command in path"를 선택한다.
   권한 정보를 입력하고 나면 화면 하단에 "shell successfully...installed in path"가 표시된다.
12.VSC를 종료하고 터미널 창에 code를 입력하면 VSC가 실행된다.
  사용자명$code
13.VSC에서 왼쪽 익스텐션 클릭하고 python을 입력하고 설치한다.
14.이제 작업 파일을 저장할 폴더를 생성해야 한다. 
   왼쪽에 "Open Folder"를 클릭해서 폴더를 원하는 위치에 만든다.
15.설치된 python 중 어떤 버젼을 사용할 것인지 지정해야한다.
   view-Command Palette...클릭하고 python interpreter를 대충 입력하고 "select interpreter를 클릭하고 원하는 버젼을 선택한다.
